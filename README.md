# Toolsman - Professional Tools & Equipment

Kenya's #1 online hardware store for professional power tools, hand tools, safety equipment and more.

## Project Overview

Toolsman is a modern e-commerce platform built with React, TypeScript, and Tailwind CSS, offering a comprehensive selection of professional tools and equipment with delivery across Kenya.

## Tech Stack

- **Frontend Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn-ui
- **Backend**: Supabase
- **State Management**: React Query (TanStack Query)
- **Form Handling**: React Hook Form
- **Routing**: React Router

## Features

- **Product Catalog**: Browse and search professional tools and equipment
- **Shopping Cart**: Add items and manage your cart
- **Multi-step Checkout**: Streamlined checkout process
- **Payment Integration**: M-Pesa payment support
- **Delivery Management**: Dynamic delivery fees based on location
- **User Accounts**: Personalized account and wishlist management
- **Responsive Design**: Mobile-friendly interface

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or bun package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/UncleBweha/toolsman.git

# Navigate to the project directory
cd toolsman

# Install dependencies
npm install
# or
bun install

# Start the development server
npm run dev
# or
bun run dev
```

The application will be available at `http://localhost:8080`

### Build for Production

```bash
npm run build
# or
bun run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

- `/src` - Source code
  - `/components` - React components
  - `/pages` - Page components
  - `/lib` - Utility functions and helpers
  - `/hooks` - Custom React hooks
  - `/styles` - Global styles

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build in development mode
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
